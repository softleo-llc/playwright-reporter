Key Features
Smart Colorized Output

Clear, color-coded results for passed, failed, skipped, and retried tests. Instantly spot issues and trends.

AI-Powered Fix Suggestions

Automatic error categorization and GenAI-powered fix suggestions for common test failures and flaky tests.

Comprehensive Metrics

Total execution time, average duration, slowest tests, pass/fail/skip stats, and more for deep insights.

Automated Email Notifications

Send beautiful summary emails with test results, build info, and links to your team after every run.

CI/CD Integration

Detects CI environments (GitHub Actions, Azure, etc.), links to builds, artifacts, and test results automatically.

Team Ownership & History

Assign tests to teams, track flaky/failing tests, and compare with previous runs for continuous improvement.

Beautiful, Actionable Reports
  • Instant summary of all test outcomes
  • Failure categorization and stack traces
  • Links to build, artifacts, and test results
  • Team ownership and flaky test detection
  • Exportable JSON for CI/CD and dashboards
How It Works
  1. Install: npm install playwright-test-reporter --save-dev
  2. Configure: Add the reporter to your playwright.config.ts with your desired options.
  3. Run Tests: Use Playwright MCP or CLI as usual. The reporter will generate rich output and JSON files.
  4. Review Results: View the summary, failures, and AI suggestions in your terminal, email, or CI dashboard.
